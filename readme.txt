Enhances Breadcrumb NavXT's support for bbPress.

== Description ==

While Breadcrumb NavXT will produce the correct breadcrumb trail for most bbPress resources, the breadcrumb trail for topic tags includes the topic archive breadcrumb rather than the desired forum archive breadcrumb. This plugin fixes that.

= Breadcrumb NavXT Versions Supported =

This plugin supports Breadcrumb NavXT 5.2.x and newer.

== Installation ==
Install via the built in WordPress plugin installer. Then activate the plugin, that's it!

To customize the breadcrumb trail you may edit the default values for the options in the administrative interface. This is located in your administration panel under Settings > Breadcrumb NavXT.

Please visit [Breadcrumb NavXT's Documentation](http://mtekk.us/code/breadcrumb-navxt/breadcrumb-navxt-doc/ "Go to Breadcrumb NavXT's Documentation.") page for more information.
